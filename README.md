# Python_ns3_simulation
> An example of a network simulation, using ns3 and Python.

## Requierments
* Python 2.7
* ns3
* matplotlib
* numpy

## Usage
#### Clone this repo
```bash
git clone https://github.com/adamruul/Python_ns3_simulation.git
```
#### Run it
```
python sim.py <Datarate in Mbps eg: 5>
```
